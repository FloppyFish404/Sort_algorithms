Comparison of time complexity of sorting methods. Generate a list of ints of desired length and find time taken to sort using different methods.

HOW TO USE:

1. Generate a list of random ints of length argv[1] to be sorted
$ python listgen.py lengthoflist

2. Run main.py to execute, and follow command line prompt
$ python main.py
'What sort method would you like to use?'

note: available sorting methods given if invalid method is suggested
